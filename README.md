# Image Uploader

A simple Image Uploader web application built using HTML, CSS, JavaScript and backend integration.

## Features

- Upload images easily
- Simple and clean UI
- Fast image handling
- Beginner-friendly project

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js / Express (if used)

## How to Run

1. Clone the repository

```bash
git clone https://github.com/0xSouravXBT/Image_Uploader.git
```

2. Open project folder

```bash
cd Image_Uploader
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
npm start
```
